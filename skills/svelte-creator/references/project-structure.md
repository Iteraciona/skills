# Project Structure Reference

Complete directory tree for a scaffolded SvelteKit + TailwindCSS 4 project.


## Route Group Conventions

| Group | Purpose | CSS File | Auth Required |
|---|---|---|---|
| `(site)` | Public marketing pages, landing, blog | `src/site.css` | No |
| `(admin)` | Protected dashboard, management pages | `src/admin.css` | Yes |

## API Proxy Naming

The api-proxy mirrors the backend endpoint structure:

| Frontend Page | API Proxy | Backend Endpoint |
|---|---|---|
| `(admin)/users/+page.ts` | `api/users/list/+server.ts` | `GET /v1/users/list` |
| `(admin)/users/[id]/+page.ts` | `api/users/[id]/+server.ts` | `GET /v1/users/:id` |
| `(site)/contact/+page.ts` | `api/contact/+server.ts` | `POST /v1/contact` |
